Localization

Localization

Localization for an entry is identified by a language code which is formed using either a ISO 639-1 language code or a combination of a ISO 639-1 language code and a ISO 3166-1 country code. Effectively, a Contensis language code can represent a language generically or specifically for a certain country.

The language code is lowercased and if the country code is included, then it is separated with a dash and uppercased.

Language code examples

  • English is represented as en
  • American English is represented as en-US
  • Germany German is represented as de-DE

Supported languages

Contensis currently supports a subset of language codes which can be set when creating a project, and enabled against a content type.
