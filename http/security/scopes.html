Scopes

Scopes

OAuth2 Scopes allow a developer to specify which resource access their application requires. These are included as a space-separated list value for the scope parameter when invoking an Authentication request.

Scope name Associated methods
Entry_Read Get Entry by id
List Entries
List Entries for content type
ContentType_Read Get content type
Project_Read Get project
